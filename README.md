# Admin Panel — Учебный проект по ИБ и Angular

Это учебный проект панели администратора, реализованный на Angular. Основной упор сделан на изучение механизмов защиты веб-приложений и разграничение прав доступа.

---

### 🛠 Технологии

* Framework: Angular 16+
* Security: Angular Route Guards (защита маршрутов)
* Data Storage: LocalStorage (клиентская авторизация)
* State: RxJS Observables

---

### 🔐 Реализация авторизации

В приложении реализована логика Client-Side Authorization. Доступ к административным функциям закрыт для неавторизованных пользователей.

**Как это работает:**

1. За доступ отвечает сервис AuthGuard, который подключен к маршруту /administration.
2. При каждой попытке перехода Guard проверяет наличие специального токена в localStorage браузера.
3. Если токен отсутствует, переход блокируется, и в консоль выводится предупреждение о нарушении прав доступа.

---

### 🧪 Инструкция по тестированию защиты

Поскольку полноценная страница логина находится в разработке, для демонстрации работы системы безопасности используется метод прямой манипуляции с хранилищем.

**1. Проверка блокировки**

Попробуйте перейти в раздел Administration. Если токен еще не создан, переход не произойдет, а в консоли появится сообщение:
> Доступ заблокирован: Требуется авторизация

**2. Получение доступа (имитация входа)**

Откройте инструменты разработчика (F12) -> вкладка Console и введите команду:

```javascript
localStorage.setItem('token', 'admin_access_token_v1');
```

После нажатия Enter и обновления страницы доступ к разделу будет открыт.

**3. Удаление доступа**

Чтобы снова закрыть доступ, удалите ключ через консоль:

```javascript
localStorage.removeItem('token');
```
