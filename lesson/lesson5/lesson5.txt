2. Выполнил Домашнее задание в котором нужно было:
	Создать type Product с полями:
	- name: string (название товара),
	- price: number (цена),
	- inStock: boolean (наличие на складе).
	Создать массив товаров (products: Product[]) из 5 элементов.
	- С помощью .filter() получить только товары, которые есть на складе.
	- С помощью .map() получить массив цен этих товаров.
	- С помощью цикла for посчитать общую сумму всех товаров в наличии.
	- С помошью .sort() отсортировать товары в наличии по цене (от дешёвых к дорогим).

2.1 Также после вывода я обнуржил некую проблему: 
	Метод sort() изменяет исходный массив A_Products, поэтому при выводе товары в наличии показывались уже в отсортированном виде.
	Решил я эту проблему следующим путем:
	Создал копию массива перед сортировкой(const Sorted_Products = [...A_Products].sort((a,b) => a.price - b.price);)
	Какой результат я получил после этого:
	A_Products сохраняет исходный порядок товаров
	Sorted_Products содержит отсортированную копию
	Примечание:
	Можно было и без копии, только нужно тогда было выводить все не в конце, а сразу после получения каждого результата
	Вывод: 
	Я считаю, что с копией код выглядит более профессионально и надежно.
