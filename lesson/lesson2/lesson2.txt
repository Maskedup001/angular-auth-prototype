1.Установил Node.js 
2.После установки установил TypeScript с помощью команды  (npm install typescript)

4.Запустил команду  npx tsc, чтобы сгенерировать базовую конфигурацию, после выполнения этой команды у мяен появился новый файл index.js, после выполнения команды и изменении кода, в нем у меня появился новый скомпилированный код
